Course images are loaded by course title.

The frontend converts a course title into a lowercase slug and looks for:

1. `<slug>.jpg`
2. `<slug>.png`
3. `<slug>.webp`
4. `<slug>.svg`

Examples for the seeded courses:

- `advanced-javascript-mastery.jpg`
- `frontend-engineering-with-react.jpg`
- `backend-apis-with-node-and-mongodb.jpg`

To change a course image, add or replace the matching file in this folder.
No homepage code change is needed.
